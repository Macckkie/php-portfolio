# Лично портфолио уебсайт (PHP)

## Описание на проекта
Този проект представлява **личен портфолио уебсайт**, разработен с помощта на **чист PHP**, HTML и CSS.  
Целта на проекта е да демонстрира основни концепции в PHP и базови умения за уеб разработка по ясен, структуриран и разбираем начин.

Уебсайтът представя информация за автора, неговите проекти и контактна форма, което го прави подходящ както за учебна задача, така и като основа за реално лично портфолио.

---

## Използвани технологии
- **PHP** – основен синтаксис, include файлове, масиви, цикли, обработка на форми
- **HTML5** – семантична структура на страниците
- **CSS3** – оформление, стилове и адаптивен дизайн
- **Без база данни** – данните се обработват чрез PHP масиви и текстови файлове

---

## Структура на проекта

php-portfolio-simple/

 index.php # Начална страница (информация за автора, умения)
 projects.php # Страница с проекти (зареждани от PHP масив)
 contact.php # Контактна форма с валидация

partials/
 header.php # Хедър (навигация и лого)
footer.php # Футър

data/
 projects.php # Данни за проектите (PHP масив)

 assets/
 style.css # Основен CSS файл
 images/
 logo.png # Лого на сайта

storage/
 messages.txt # Записани съобщения от контактната форма

 README.md


---

## Основни функционалности
- Повторно използване на код чрез `require` (header и footer)
- Динамично зареждане на проекти от PHP масив
- Контактна форма с:
  - Валидация на входните данни
  - Обработка на грешки
  - Запис на съобщенията в текстов файл
- Кликваемо лого, което води към началната страница
- Изчистен и минималистичен дизайн
